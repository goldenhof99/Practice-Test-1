Old Fashion vs Desktop Publishing

Technologies Used

For old fashioned typesetting it involved a lot of manual labour and using equipment such as letterpresses. While HTML uses softwares such as github that allow for much more efficent labour.

Over Design Controls

Old fashioned typesetting were greatly limited on their physical tools and required heavily on the print process. While HTML and CSS can provide live edits and can go back and change mistakes far easier.

Visuals

The limited resources of the old fashioned caused for perhaps a lack of outstanding visual creativity, while HTML and CSS can create modern visuals as scene on webpages and docuemnts like the one you are reading.
